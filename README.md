# 🖤 Ruru Finds

A minimal, aesthetic affiliate marketing website designed to showcase curated fashion and lifestyle products.

Live the scroll. Trust the vibe.

## ✨ About

Ruru Finds is a curated space where fashion, art, creativity, and tech come together.

I’m a 4th year at IIT Kharagpur, mixing code, creativity, and curated finds into one aesthetic corner. Every product here is something I’d actually wear, obsess over, or impulsively add to cart at 2am. No clutter, no randomness, just pieces that feel right.

## 🚀 Features

- Minimal black, pink, and gold aesthetic  
- Pinterest-style masonry layout  
- Category-based product filtering  
- Reusable product cards  
- Smooth scrolling and subtle animations  
- Affiliate link integration  
- Fully responsive (mobile-first)

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

No frameworks, just clean frontend.

## 🧠 How to Add Products

All products are managed inside script.js.

Add a new product like this:

{
  name: "Product Name",
  category: "Category",
  image: "assets/image.jpg",
  link: "https://affiliate-link.com",
  desc: "short caption"
}

Categories are generated automatically.
